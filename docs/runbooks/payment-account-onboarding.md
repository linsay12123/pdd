# 支付账号开通手册

这份手册只做一件事：把你需要亲自去注册的支付账号，按顺序拆开。

原则很简单：

- 能由我在代码里先准备的，我先准备
- 必须由你本人去实名、上传证件、绑定银行卡的，我会停在那一步等你
- 我们一次只办一个，避免你同时开太多后台把自己搞乱

## 推荐顺序

1. `支付宝`
2. `微信支付`

原因：

- `支付宝` 和 `微信支付` 都涉及国内商户资料，通常更花时间
- 稳定币这条你已经决定走人工确认，而且地址我已经先写进系统了
- 你现在不做 `Stripe`，所以不需要再去开 Stripe 商家账号

## 支付宝

你要做的事：

1. 注册支付宝开放平台商家账号
2. 完成商户认证
3. 开通你要用的支付能力
4. 后面把商户参数给我

通常要准备：

- 主体证件
- 收款银行卡
- 联系人信息
- 商户资料

我这边已经准备好的：

- 支付宝下单接口骨架
- 支付宝异步通知入口

等你开通好，我再帮你把真实参数接进去。

## 微信支付

你要做的事：

1. 注册微信支付商户平台账号
2. 完成商户审核
3. 开通对应支付产品
4. 后面把商户参数给我

通常要准备：

- 主体证件
- 结算银行卡
- 联系人信息
- 商户资料

我这边已经准备好的：

- 微信下单接口骨架
- 微信支付回调入口

等你开通好，我再把真实参数接进去。
